# 家庭資產管理系統 v2.0

單純記錄四人資產數值：萱、憲、傑、文。

## 功能
- 新增 / 修改 / 刪除每日資產
- 自動計算總資產、每日增減、月總計、年總計
- 黑金尊爵版 / 招財綠金版切換
- 手機與電腦可用
- 歷史紀錄、月曆顯示、資產走勢圖
- CSV / Excel 匯出與 CSV 匯入

## Streamlit Cloud 部署
1. 將此資料夾上傳到 GitHub repository
2. 到 Streamlit Cloud 建立新 App
3. Repository 選你的 GitHub 專案
4. Main file path 填：`app.py`
5. Deploy

## 手機加入桌面
- iPhone：Safari 開啟網址 → 分享 → 加入主畫面
- Android：Chrome 開啟網址 → 右上角選單 → 加到主畫面

## 注意
免費 Streamlit Cloud 的本機 `data.csv` 不是正式資料庫，建議定期用匯出功能備份。
